Private Commands

owneractives

Show active streams in all groups

ownerstop <short_id>

Stop a stream by short id

ownerstream <short_id> <name>

Start streaming in selected group

ownerchange <short_id> <name>

Change stream in selected group

ownerquality <short_id> <low|medium|max>

Change quality

ownerauto <short_id> <seconds>

Set auto-delete

ownertitle <short_id> [new_title]

View or change title

ownerchats

Show groups and short ids

ownerstate <short_id>

Show live state for short id
