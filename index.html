Stream Bot Commands

stream <name>

Start a saved channel

change <name>

Switch to another channel

stop

Stop the live stream

list

Show saved streams or tables

title <name>

Set the live title

quality <low|medium|max>

Set stream quality

auto <seconds>

Set auto-delete timer

owners add <table> <name> <cdn>

Add a stream

owners delete <name>

Remove a stream

owners reply grant

Grant special access

owners reply revoke

Revoke special access

specials

Show owners & special users

state

Show live state for this chat

comment <Allow|Disallow>

Enable or disable comments

help
